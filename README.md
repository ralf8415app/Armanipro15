# Chanel & Armani – kompletter Neubau

Diese Version wurde technisch neu erstellt und übernimmt keinen alten Spielzustand, keinen alten Service Worker und keinen alten Cache.

Enthalten:
- vier Länder-Level: Italien, Frankreich, Japan, Brasilien
- Inseln, Wasser und begehbare Berge
- Palmen-Schwingen mit Erklärung nur an der ersten Palme in Level 1
- Palmen auch im Boss-Level
- Boden-Gegner patrouillieren hin und her und bleiben auf Plattformen
- fliegende Gegner besitzen Flügel
- böser Bären-Endboss
- Doppelsprung
- Weltraum-Bonuslevel mit Chanel/Armani-Raumschiff, Tiergegnern im Retro-Stil und großem Boss
- responsive Steuerung für iPhone, iPad, Android und Windows
- keine Mario-Röhren und kein Ladebalken

## Installation als iPhone-Home-App
Die Dateien über GitHub Pages oder einen anderen HTTPS-Webserver bereitstellen. Danach in Safari öffnen und „Zum Home-Bildschirm“ wählen. Die ZIP-Datei allein aus der Dateien-App kann keine PWA installieren.
