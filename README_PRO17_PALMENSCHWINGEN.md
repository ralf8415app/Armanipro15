# Chanel & Armani – Pro 17 Testversion

Neu in dieser Testversion:

- Große Palmen auf allen Hauptinseln
- Armani oder Chanel können sich mit einem Seil von den Palmen schwingen
- Touch-Taste 🕸️ sowie Tastatur E/Shift
- Links/rechts erhöht den Schwung
- Sprungtaste oder erneutes 🕸️ lässt die Figur los
- Schwung kann genutzt werden, um über Gegner und Wasserlücken zu gelangen

Start: `index.html` öffnen. Auf Mobilgeräten Querformat verwenden.
