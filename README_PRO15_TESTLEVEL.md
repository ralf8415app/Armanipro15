# Chanel & Armani – Pro 15 Testlevel

Dieses Paket basiert auf dem vollständigen Pro-12-Grundspiel und enthält ein echtes Testlevel.

## Testlevel
- sechs große Inselabschnitte mit sichtbaren Wasserlücken
- begehbare Bergketten mit Kollision
- glatte Inselunterseiten ohne Zacken
- Boden-Gegner ausschließlich auf Inseln und Bergen
- fliegende Gegner ausschließlich als Fledermäuse
- weiter herausgezoomte Handyansicht
- Sammelobjekte und Test-Endgegner

Start: `index.html` öffnen oder das Paket bei GitHub Pages hochladen.
